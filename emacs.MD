(SPACE)Emacs tips and tricks
------------------------

# Buffers

- New buffer: `SPC b N` (meta buffer new)
- Copy whole buffer: `SPC b Y` (meta buffer Yank)

# Macros

- Record macro (evil-mode): `q a`
- Stop recording macro: `q`
- Run last macro: `@ a`
- Run macro 100 times: `100 @ a` (similar to `C-u 100 @ a`)
